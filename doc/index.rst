tmuxp
=====

Welcome to tmuxp's documentation. tmuxp is a tool for managing
:term:`tmux(1)` workspaces.

tmuxp handles:

- launching from inside or outside a tmux client
- freezing tmux session configs
- export / import configs in JSON, YAML, INI
- a clean, python API.
